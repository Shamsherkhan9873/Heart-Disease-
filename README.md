# Heart-Disease-
Data visualization project using Tableau to explore heart disease risk factors such as BMI, smoking, age, and physical health.
# Heart Disease Analysis using Tableau

## Project Overview
This project presents an interactive Tableau dashboard that analyzes heart disease data to identify important health risk factors. The dashboard transforms raw healthcare data into meaningful visual insights.

The objective of this project is to understand patterns in heart disease cases and explore how lifestyle and medical factors influence heart disease risk.

## Dataset
The dataset used in this project contains 4500 records with 18 health-related attributes.

Some of the important variables include:
- BMI (Body Mass Index)
- Smoking
- Alcohol Drinking
- Stroke
- Physical Health
- Mental Health
- Difficulty Walking
- Sex
- Age Category
- Diabetic
- Physical Activity
- Asthma
- Kidney Disease
- Skin Cancer
- Heart Disease

## Tools Used
- Tableau
- CSV Dataset
- GitHub

## Dashboard Features
The Tableau dashboard includes several visualizations such as:
- Heart disease distribution
- Age category analysis
- BMI and health risk comparison
- Smoking and alcohol impact on heart disease
- Physical activity and health condition insights

Interactive filters allow users to explore the data dynamically.

## Objective
- Analyze heart disease data
- Identify major health risk factors
- Create interactive visualizations
- Present insights through a Tableau dashboard

## Conclusion
The analysis shows how lifestyle habits and health conditions may influence the probability of heart disease. The dashboard provides a simple and interactive way to explore these relationships.
